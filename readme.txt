Proyecto de e-commerce de facil navegación y entorno amigable, con presentación de los productos, gestión de stock y posibilidad de recopilación de datos de los usuarios para un contacto posterior. Cuenta con armado de órdenes de compra a través de Firebase de Google, lo cual permite el análisis de las operaciones y armado de estadísticas para una mejor gestión.
Cuenta con la incorporación de librerías Sweet Alert y Bootstrap para lograr una interación simple y atractiva para el usuario, que podrá acceder desde cualquier dispositivo y recibirá notificaciones emergentes de confirmación de sus acciones.
